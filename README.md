# Docker-Apache2

A simple Apache2 file-server running on Alpine container

# Usage

- Ingredients
  - Linux

    ```console

    foo@bar:~$ source envsetup.sh

    ```

  - Windows

    ```console

    foo@bar:~$ envsetup.bat

    ```

- Just, fire it up !

  ```console

  foo@bar:~$ docker compose up -d

  ```

- Or, put it down ...

  ```console

  foo@bar:~$ docker compose down

  ```
